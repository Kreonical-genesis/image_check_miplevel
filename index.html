What is Mipmap?
Only block/item atlas affected (terrain.png, items.png). GUI uses separate atlas.

How it works: Creates downscaled copies (÷2 each level) for distant objects. Reduces aliasing & improves FPS.

Vanilla max: 4 levels. If any texture can't create full chain, mip level drops for ALL textures (see log: "limits mip level").

Hi-res packs (256x+): May need mods (Better Mipmaps) to add more levels.
